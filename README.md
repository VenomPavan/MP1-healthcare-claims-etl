# Healthcare Claims ETL

## Overview

A Healthcare Claims ETL pipeline built using Python, Pandas, SQLite, and SQL.

## Project Flow

claims.csv
↓
Extract
↓
Transform
↓
Load
↓
SQLite Database
↓
SQL Reporting

## Features

- Data Extraction
- Data Cleaning
- Data Validation
- Exception Handling
- Logging
- SQLite Storage
- SQL Reporting

## Technologies Used

- Python 3.11
- Pandas
- SQLite
- VS Code

## Run ETL

python main.py

## Run Report

python report.py
